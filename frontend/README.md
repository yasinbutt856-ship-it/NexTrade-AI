# NexTrade AI — Frontend

React 19 SPA for the trading platform. Built with TypeScript, Tailwind v4, Recharts, and React Query.

## Commands

```bash
npm run dev      # Development server (port 5173)
npm run build    # TypeScript check + production build
npm run preview  # Preview production build
```

## Environment

```env
VITE_API_URL=https://your-backend.com
```

Deploys to Netlify automatically from `frontend/` directory.
